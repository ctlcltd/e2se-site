Edit Transponders

Edit transponders and transponder parameters.

First you could read Quick Start page of online help for a quick introduction, with basic notions: context menu, Drag and Drop, multiple selection.

How to edit

Select the transponder to edit, open the context menu and click on "Edit Transponder".

In the dialog, edit the transponder parameters, then press "Save".

Transponder can be quickly edited by double-clicking on the selected service channel.

Note: Transponder parameters will be changed for any service associated with the transponder.

Properties

    From Transponder group you will change the following transponder properties:

  • Type, type of service: Satellite, Terrestrial, Cable, ATSC
  • Transport ID, Transport Stream identifier
  • DVBNS, DVB namespace identifier from Enigma 2 specs
  • ONID, network identifier
  • Position, orbital position of the transmitter (Satellite) expressed in degrees and W/E suffix: W = west, E = east, equal to 0.00 W/E for: Terrestrial, Cable, ATSC

Parameters [Type]

Editable parameters vary depending on the Type selected in Transponder group.

Parameters for Satellite
  • Frequency, expressed in KHz
  • Polarization, with values: H = horizontal, V = vertical, L = circular left, R = circular right
  • Symbol Rate, expressed in bit/s
  • FEC, acronym for Forward Error Control
  • System, with values: auto = automatic selection, DVB-S, DVB-S2, DVB-S/S2
  • Modulation
  • Inversion
  • Roll offset
  • Pilot

extended:

  • isid
  • mts
  • plsmode
  • plscode
  • plsn
Parameters for Terrestrial
  • Frequency, expressed in KHz
  • Constellation, synonym for modulation
  • Bandwidth
  • HP FEC, acronym for High-Pass Forward Error Control
  • LP FEC, acronym for Low-Pass Forward Error Control
  • Transmission Mode
  • Inversion
  • Guard Interval
  • Hierarchy

(Lamedb only):

  • System, with values: auto = automatic selection, DVB-T, DVB-T2
Parameters for Cable
  • Frequency, expressed in KHz
  • Modulation
  • Symbol Rate, expressed in bit/s
  • Inner FEC, acronym for Forward Error Control
  • Inversion
Parameters for ATSC
  • Frequency, expressed in KHz
  • Modulation
  • System, with values: empty = automatic selection, ATSC, DVB-C ANNEX B

Edit from channel

Select the service channel to edit, open the context menu and click on "Edit Service".

Service can be quickly edited by double-clicking on the selected service channel.

From Tuner group you will change the transponder:

  • Satellite, position selection
  • Transponder, transponder selection
  • to create a transponder, press "New Transponder"

To change the transponder parameters, go to Transponder tab.

Edit the transponder parameters, then return to Service tab or press "Save".

The service will be changed in every position where found.

Note: The transponder parameters will be changed for any service associated with the transponder.

For channel lists guide follow Channel List operations page of online help.

For transponders guide follow Transponder operations page of online help.
