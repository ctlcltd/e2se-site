Transponder operations

[INTRODUCTION]

Move, modify, delete transponders.

Move, edit, delete positions [XML only].

Import, export settings.

First you could read Quick Start page of online help for a quick introduction, with basic notions: context menu, Drag and Drop, multiple selection.

Transponder sorting

Transponders can be sorted according to the column name (e.g. sorted by Frequency 0-9 or Frequency 9-0) in ascending and descending order.

To use the sort click on column name in the tree view, inside the Transponders area.

Move transponders

The transponders move with Drag and Drop, or alternatively with the Cut - Copy - Paste functionality.

Select transponders then drag them with the mouse cursor to the desired position.

For multiple selection press Ctrl on keyboard, or ⌘ command on a Mac keyboard, for continuous selection press ⇧ Shift on keyboard, and select services, channels, markers with the mouse or touchpad, then drag them into transponder item.

[XML only] Note: It is not possible to move transponders to other positions (satellites).

Create transponder

To create a transponder, click on "+ New Transponder".

In dialog, enter the transponder parameters, then press "Save".

[XML only] The transponder will be added at the current position.

For further information, view Edit Transponders page of online help.

Edit transponder

Select the transponder to edit, open the context menu and click on "Edit Transponder".

In dialog, edit the transponder parameters, then press "Save".

For further information, view Edit Transponders page of online help.

Transponder can be quickly changed by double-clicking on selected service channel.

Note: Transponder parameters will be changed for any service associated with the transponder.

Delete transponder

To delete a transponder, select the transponders, open the context menu and click on "Delete".

[XML only] The transponder will be deleted from the current position.

Transponder can also be deleted with keyboard by pressing on ⇤ Backspace, Del, Delete, depending on system and keyboard used.

Note: The services associated with the transponder will be orphaned.

Edit settings [XML only]

To edit the settings, click on icon in the Positions area.

In dialog, modify the settings properties, then press "Save".

Move positions [XML only]

Positions move with Drag and Drop.

Select the position then drag it with the mouse cursor to the desired position.

Note: Multiple selection is not available to move positions.

Edit position [XML only]

Select the position to edit, open the context menu and click on "Edit Position".

In dialog, edit the position properties, then press "Save".

The position can be quickly changed by double-clicking on selected position.

Create position [XML only]

To create a position, click on "+ New Position".

In dialog, enter the position properties, then press "Save".

Import and export [XML only]

To import and export settings.

Import

From quick menu, click on "Import".

From main menu, click menu File > Import menu.

In File dialog select the settings file to import.

Export

From quick menu, click on "Export".

From main menu, click menu File > Export menu.

In File dialog select the location where you want to export settings.
