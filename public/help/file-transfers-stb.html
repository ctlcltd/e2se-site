Transfers from and to Decoder

How to transfer files from and to Decoder (STB).

Handle connection profile, add and remove profiles.
Connect and disconnect the software to STB.
Download and Upload channel lists from / to software from / to STB.

First you could read Quick Start page of online help for a quick introduction, with basic notions: context menu, Drag and Drop, multiple selection.

Connection profiles

From quick menu, select the desired connection profile in the dropdown menu.

This is usually "Default" profile.

To add and remove connection profiles, open "Settings".

From quick menu, click on "Settings"

From main menu, click menu File > Settings, or on Mac click menu "e2 SAT Editor" > Preferences…

Connect and Disconnect

Select the desired connection profile, usually "Default" profile.

Connect

From quick menu, press "Connect"

The software will attempt a connection to Decoder (STB).

Disconnect

From quick menu, press "Disconnect"

The software will attempt a connection to Decoder (STB).

Download files

Select the desired connection profile, usually "Default" profile.

Open a channel list or create a new one by clicking on "+ New Tab" at top window.

When use an existing channel list, downloaded lists will merged with pre-existing channels.

From quick menu, press "Download"

The software will attempt a connection to Decoder (STB).

The channel lists will be downloaded and displayed in the software.

For channel lists guide follow Channel List operations page of online help.

Upload files

Select the desired connection profile, usually "Default" profile.

Open an existing channel list.

From quick menu, press "Upload"

The software will attempt a connection to Decoder (STB).

The channel lists will be loaded into the STB.

Therefore, the software will attempt to update channel lists on STB.

Now STB will display fresh loaded channel lists.

Note: the software does not currently upload / download picons, these must be uploaded manually to the STB.

For channel lists guide follow Channel List operations page of online help.

Troubleshooting

For problems when transferring files via FTP, refer to Troubleshooting.
