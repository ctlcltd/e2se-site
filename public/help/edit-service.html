Online Help

Edit Services

Edit service channels and service parameters.

Properties, parameters, tuner, transponder, package, PID, Card ID, flags.

First you could read Quick Start page of online help for a quick introduction, with basic notions: context menu, Drag and Drop, multiple selection.

How to edit

Select the service channel to edit, open the context menu and click on "Edit Service".

In dialog, edit the service parameters, then press "Save".

The service will be changed in every position where found.

Service can be quickly edited by double-clicking on selected service channel.

Properties

From Service group you will change the following service properties:

  • Service name, name of the service channel
  • Service type, to enter a custom numeric type press "Custom"
  • Service ID, service identifier

Transponder

From Tuner group you will change the transponder:

  • Satellite, position selection
  • Transponder, transponder selection
  • to create a transponder, press "New Transponder"

To change the transponder parameters, go to Transponder tab.

Edit the transponder parameters, then return to Service tab or press "Save".

For further information, view Edit Tranponders page of online help.

Note: The transponder parameters will be changed for any service associated with the transponder.

Parameters

From Parameters group you will change service parameters:

  • Package, contains provider name
  • PIDs, contains service PID parameters, eg. VPID, APID, PCRPID, ...
  • CAIDs, contains Card ID parameters of the service in comma-separated-value, eg. C:0100,C:0200
  • Flags, contains service flags, eg. Flag as new service, ...
  • Custom, to raw edit service parameters in comma-separated-value, eg. p:ProviderName,c:0100,C:0200
  • Extras, contains extra parameters: Service Number, Src ID

For channel lists guide follow Channel List operations page of online help.

For transponders guide follow Transponder operations page of online help.
