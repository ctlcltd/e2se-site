Online Help

Settings

How to change settings, handle connection profiles and engine.

Connections, contains connection profiles and connection parameters to Decoder STB.
Preferences, contains appearance and behavior preferences.
Engine, contains behavior preferences of the contained engine.
Advanced, contains access to advanced settings.

First you could read Quick Start page of online help for a quick introduction, with basic notions: context menu, Drag and Drop, multiple selection.

Settings

Software preferences are found in "Settings".

From quick menu, click on "Settings"

From main menu, click on File > Settings, or on Mac click menu "e2 SAT Editor" > Preferences….

A dialog will open, to save settings press "Save", to cancel "Cancel".

Note: Not all the settings are changed real-time, in some cases it might necessary to restart the software.

Connection settings

Connections tab contains all the settings for connecting to Decoder (STB).

To save settings press "Save", to cancel "Cancel".

Profiles

Connection profiles are found in the side panel.

To create a new connection profile, press on "+", then name the profile.

To remove a connection profile, press on "–".

By double-clicking on the item panel you can rename the profile.

Profile properties

Select a profile from the side panel.

The software automatically creates a profile with the most common settings, named "Default".

Connection

Connection group contains the following connection parameters to STB:

  • IP Address, IP address of STB connected to the network
  • FTP Port, FTP connection port to STB, eg. 21
  • Use Active FTP, on / off, set the FTP connection as FTP Active instead of FTP Passive
  • HTTP Port, Webif connection port to STB, eg. 80
Login

Login group contains the following connection parameters to STB:

  • Username, FTP / Telnet login username of STB
  • Password, FTP / Telnet login password of STB
Configuration

Configuration group contains the following properties for interfacing with STB:

  • Transponders, path containing XML settings (satellites.xml, ...) related to transponders, eg. /etc/enigma2/
  • Services, path containing service channel lists (lamedb, blacklist, ...), eg. /etc/tuxbox/
  • Bouquets, path containing groups of channels (userbouquet.dbe01.tv, bouquet.tv, ...), eg. /etc/enigma2/
  • Picons, path containing channel logo (picons), eg. /usr/share/enigma2/picon/
Commands

Commands group contains the following advanced properties for interfacing with STB:

  • Custom webif reload URL address, URL address parameter for updating the STB lists, eg. /web/servicelistreload?mode=0
  • Custom telnet reload command, Telnet command parameter for updating the STB lists, eg. init 3

Preferences settings

Preferences tab contains all the settings related to the appearance and behavior of the software.

To save settings press "Save", to cancel "Cancel".

General

General group contains the following preferences:

  • Show confirmation messages when deleting, on / off
  • Parental lock whitelist (exclusion instead inclusion list), on / off
Language

Language group contains language settings.

System detected language is selected by default.

  • "Default (system language)", auto, selected by default
  • ...
Theme

Theme group contains settings related to the appearance of the software.

  • "Default (system theme)", auto, selected by default
  • "Dark" variant, dark theme
  • "Light" variant, light theme
Drag and Drop

Drag and Drop group contains settings related to Drag and Drop behavior.

  • Switch to current bouquet item after the drop, on / off

Channel operations, choice:

  • Copy channels (preserving)
  • Move channels (deleting)
Picons editor

Picons editor group contains settings related to the behavior of Picons Editor.

  • Backup picon when replaced, on / off

Filename format, choice:

  • Use reference ID, will generate picon using Reference ID, eg. 1_1_0_10_1234_13E_820000_0_0_0.png
  • Use service name, will generate picon using channel Service Name, eg. ribasports.png

Engine settings

The "Engine" tab contains all preferences related to the behavior of the engine contained in the software.

To save settings press "Save", to cancel "Cancel".

Settings

Settings group contains preferences related to settings.

Select a default format with which new channel lists will be created.

"Default format" selector contains these options:

  • Lamedb 2.4 [Enigma 2]
  • Lamedb 2.5 [Enigma 2]
  • Lamedb 2.3 [Enigma 1]
  • Lamedb 2.2 [Enigma 1]
  • Zapit api-v4 [Neutrino]
  • Zapit api-v3 [Neutrino]
  • Zapit api-v2 [Neutrino]
  • Zapit api-v1 [Neutrino]
Tools

Tools group contains preferences related to tools: import and export.

  • CSV Import/Export, preferences related to CSV
  • Fields Import/Export, preferences related to fields to import / export

Advanced settings

"Advanced" tab gives access to the advanced settings of the software.

A cautionary warning will be shown, you should [confirm] to access advanced settings.

Recommendation: Do not do this if you do not understand what you are doing.

In the editable table, double-click on the cell in the VALUE column to change the value.

For boolean values (TRUE or FALSE), input text: true or false.

To save the settings press "Save", to cancel "Cancel".

Reset settings

The software can be reset to factory settings.

All settings previously saved will be erased.

Recommendation: Do not do this if you do not understand what you are doing.

To reset to factory settings, go to "Settings" > Advanced > [confirm].

Change the settings.reset line from FALSE to TRUE, typing true.

Save your settings by pressing "Save" and close the software.

The next time the software is opened it will be reset to factory defaults.

To manually edit the settings file, please refer to the Qt toolkit documentation at: https://doc.qt.io/qt-6/qsettings.html#platform-specific-notes

The software uses the identifier "io.github.ctlcltd.e2se".
