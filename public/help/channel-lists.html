Online Help

Channel List operations

[INTRODUCTION]

Create, edit, import, export, convert, channel lists.

Move, edit, delete and lock channels.
Move, modify, delete bouquets (group of channels).

First you could read Quick Start page of online help for a quick introduction, with basic notions: context menu, Drag and Drop, multiple selection.

Create channel lists

To create a new list of channels, open a new tab by clicking on "+ New Tab" at top window.

From main menu, click menu File > New.

Edit channel lists

Open the channel list.

From main menu, click menu File > Open.

From quick menu, click on "Open".

Save channel lists

From main menu, click menu File > Save, or to save with another name or to another location, click menu File > Save as.

From quick menu, click "Save".

You may see a message about overwriting of existing files, choose "Save" or "Cancel".

Convert channel lists

Channel lists can be converted between formats, from Enigma 2 to Neutrino and from Neutrino to Enigma 2.

From the "Services" side list above All services, click on the [IMG] icon.

Then from the drop-down menu, press Convert menu item, select the option responding to the desired conversion.

Available from / to conversion options are:

  • Lamedb 2.4 [Enigma 2]
  • Lamedb 2.5 [Enigma 2]
  • Lamedb 2.3 [Enigma 1]
  • Lamedb 2.2 [Enigma 1]
  • Zapit api-v4 [Neutrino]
  • Zapit api-v3 [Neutrino]
  • Zapit api-v2 [Neutrino]
  • Zapit api-v1 [Neutrino]

Import and export

To selectively import and export parts of channel lists and merge with the opened channel list, use Import and Export functionality.

Import

From quick menu, click on "Import".

From main menu, click menu File > Import.

In File dialog, select file or directory to import. In dialog you can select the type of file to import from File type selector.

Export

The current selected service channel list or bouquet will be exported.

From quick menu, click on "Export".

From main menu, click menu File > Export.

From bouquet you want to export, open the context menu and click on "Export".

In File dialog, select the location where you want to export. In dialog you can select the type of file to export from the File type selector.

Channel sorting

Channels can be sorted according to the column name (e.g. sorted by Name A-Z or Name Z-A) in ascending and descending order.

To use the sort click on the column name in the tree view, inside the Channels area.

When using channel sort, Drag and Drop is not available, to quickly re-enable Drag and Drop click on the "Drag&Drop" button.

Move channels

Channels move with Drag and Drop, or alternatively with the Cut - Copy - Paste functionality.

Select services, channels, markers, then drag them onto the Bouquets area with the mouse cursor, into the bouquet item.

For multiple selection press Ctrl on the keyboard, or Command on the Mac keyboard, for continuous selection press ⇧ Shift on the keyboard, and select services, channels, markers with the mouse or touchpad, then drag them into channel item or bouquet item onto the Bouquets area.

When using channel sort, Drag and Drop is not available, to quickly re-enable Drag and Drop click on the "Drag&Drop" button.

View channel details (Reference)

Further channel details are available by pressing the "Reference" button.

The details of the selected service channel will be shown in a box:

  • Reference ID, reference identifier
  • Service ID, service identifier
  • Transponder, transponder information, associated with the service
  • Userbouquets, the channel group where the service is
  • Bouquets, the parent group of channels, TV or Radio, where the service is
  • Tuner, tuning information, eg. satellite, position

Lot of details can be found in Edit Service dialog, which can be reached from the contextual menu by double-clicking on the selected service.

  • When editing Service, these parameters will be shown: PID, Card ID, Flag, Service Number, Src ID.
  • When editing Transponder, all the transponder parameters associated with the service will be shown.

Add channels

To add channels click on "+ Add Channel".

The channel selector, with filters, will be shown in the pop-up window.

Select the channels to add and press "Add".

When you done, close the pop-up window.

Note: Channels cannot be added to the Services lists: All services, TV, Radio, Data.

Make sure you have selected a bouquet in the Bouquets area or have created one and select it.

Edit channel

Select the service channel to edit, open the context menu and click on "Edit Service".

In the dialog, edit the service parameters, then press "Save".

The service will be modified in every position where found.

For further information, view Edit Services page of online help.

Service can be quickly edited by double-clicking on the selected service channel.

Delete channel

To delete a channel, select the channels, open the context menu and click "Delete".

The service channel will be deleted in the current bouquet.

Channel can also be deleted with keyboard by clicking on ⇦ Backspace, Del, Delete, depending on system and keyboard used.

Create service channel

To create a service channel, click "+ New Service".

In the dialog, enter the service parameters, then press "Save".

The service channel will be added into the current bouquet.

For further information, view Edit Services page of online help.

Create marker

To create a marker, click "+ Add Marker".

In the dialog, enter a name for marker, then press "Save".

The marker will be created in the current bouquet.

Note: Markers cannot be created in the Services lists: All services, TV, Radio, Data.

Make sure you have selected a bouquet in the Bouquets area or have created one and select it.

Edit marker

Select the marker to edit, open the context menu and click on "Edit Marker".

In the dialog, change the name of the marker, then press "Save".

The marker will be changed in the current bouquet.

Marker can be quickly edited by double-clicking on the selected marker.

Delete marker

To delete a marker, select markers, open the context menu and click "Delete".

The marker will be deleted from the current bouquet.

Marker can also be deleted with keyboard by pressing on ⇦ Backspace, Del, Delete, depending on system and keyboard used.

Move bouquets

The bouquets move with Drag and Drop.

Select bouquets then drag them with the mouse cursor to the desired position.

Note: Multiple selection is not available for moving bouquets.

Create bouquet

To create a bouquet, click on "+ New Bouquet".

In the dialog, enter a name for bouquet, then press "Save".

The bouquet will be created below the current parent group of channels, TV or Radio.

Edit bouquet

Select the bouquet to edit, open the context menu and click on "Edit Userbouquet".

In the dialog, change the name of the bouquet, then press "Save".

Bouquet can be quickly edited by double-clicking on the selected bouquet.

Delete bouquet

To delete a bouquet, select the bouquet, open the context menu and click "Delete".

The bouquet will be deleted with all the contained channels.

Lock channels (Parental)

The Parental lock functionality is available with both blacklist and whitelist, this preference is found in "Settings" > Preferences.

Single channel lock

To lock a channel, select the channel, open the context menu and click on "Set Parental lock".

To unlock a previously locked channel, select the channel, open the contextual menu and click on "Unset Parental lock".

Note: Multiple selection is not available to lock channels.

Channel group lock

Bouquets can be created with channels to lock [blacklist] or unlock [whitelist].

Create a bouquet with the channels to lock/unlock.

Select the bouquet in the Bouquets area, open the context menu and click on "Set Parental lock".

To restore service channels to their previous state, open the context menu and click on "Unset Parental lock".
