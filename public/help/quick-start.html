Online Help

Quick start

Quick start guide for using the software.

The software features can be accessed from the quick menu at the top, and from the main application menu
(File, Edit, Find, Tools, Window, Help).

Quick menu

The quick menu items:
Open, Save, Import, Export, Settings, Connection profile selection, Connect, Disconnect, Upload, Download.

  • To open and save channel lists: Open, Save.
  • To import and export channel lists, individual channel groups (bouquet and userbouquet), settings (eg. "satellites.xml"): Import, Export.
  • For software settings, connection profiles, engine, advanced settings: Settings.
  • To select connection profile: drop-down menu (eg. Default).
  • To control the connection, upload and download files via FTP from the Decoder STB: Connect, Disconnect, Upload, Download.

Editing functions: [item] right-click

To reach the editing functions of [item] (channels, services, transponders, markers) use the context menu.

The context menu can be reached by right-clicking on the selected item (highlighted) with the button right of the mouse or touchpad.
Some keyboards also have a special key to call up the context menu.

Main editing function: [item] double-click

The main editing function of [item] (channels, services, transponders, markers) can be reached by double-clicking on the selected item (highlighted).
For example: double-clicking on the channel will open "Edit Service" dialog window of the corresponding channel item.

Move channels: [item] Drag and Drop

The [item] (channels, services, transponders, markers) can be moved by Drag and Drop, or using Cut - Copy - Paste.

The item(s) can be moved from the channel group (userbouquet) being viewed to another by dragging the selected channel item (highlighted) over the desired channel group (userbouquet) item.

Multiple selection: [items]

You can select multiple [items] (channels, services, transponders, markers) with multiple selection.

Multiple selection can be done by pressing the Ctrl key and the Shift key on the keyboard, or command and Shift on a Mac keyboard.

Handle connection profiles: Settings

The connection settings are found in Settings, reachable by pressing Settings on the top quick menu, or from Settings and Preferences… depending on the system you are using from the main application menu.

For more information on using the mouse, touchpad, keyboard, context menu, multiple selection, Drag and Drop, please refer to the instruction manual of the device or system you are using.
