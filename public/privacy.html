Below are technical informations and of the nature of exchanged data of this Website, between machines and User devices.

Website logs

Websites, for technical reasons, law obligations and/or for protection, must necessarily record visits. Whenever you visit the Website, information about the visit will be stored for limited time: date, time, IP address, browser used and its characteristics. The Website uses the HTTPS security protocol, with SSL certificate, for security within communication. Information on the visit could be sent exceptionally also to the certifying authority that manages the issue of the SSL certificate, this feature of the User's browser is known as "OCSP Responder".

These are the third parties involved in the communication between the Website and the User's browser:

Hosting:
HostingPerTe, Italy, hostingperte.it
SSL certificate:
ISRG (Internet Security Research Group), USA, letsencrypt.org

Website and browser storage

The Website uses the storage available on the User's browser to store the color preference (light and dark color variants) and to operate within the Translations app. The Translations app allows users who reach it at this address: https://e2sateditor.com/translate/, to provide more accurate translations for the Software. In detail, this Website uses the WebStorage API functionality provided by the browser, the translation data is temporarily stored on the User's device in order to be modified and subsequently sent.

This temporary data is automatically deleted when the User's browser is closed.

This data is NOT traceable and is NOT used to track anything.

Third Parties involved in the Software

The Software uses third-party, free software and/or open-source libraries.

"Qt" toolkit for the gui and "libcurl" library for the connectivity (FTP, telnet, Webif).

There are no known uses of User data by these libraries.
